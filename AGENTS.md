Novas-c can be downloaded from https://ascl.net/assets/codes/NOVAS/novasc3.1.zip. The code should not be distributed in this repo, but fetched (and cached) by the build.rs.

This repository is a Rust-binding to it.

It has the goals of building to standard and WASM targets.

It should keep its own code minimal.

For compatibility purposes, there can be patches applied on the C code to make this work.

The readme should be engaging but serious, emphasizing the correctness of the underlying source code. It should always be kept up-to-date. The readme should provide attribution for the original developers. It should state that this binding was entirely generated by AI, but no AI code is included in the final binary as it's solely based on the C source.
